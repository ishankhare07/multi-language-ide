Welcome to multi-language-ide.

This is the home page of the multi-language-ide project

Get it on your machine by following the instructions below:

$ git clone git@github.com:ishankhare07/multi-language-ide.git
$ cd multi-language-ide
$ python3 main.py

If you're using the GitHub for Mac, simply sync this repository and you'll see the new folder.
